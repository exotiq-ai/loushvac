> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a complete, lead-generating Astro static site for Lou's Heating & Cooling (Denver HVAC company) covering all 9 stages in the CLAUDE.md spec — asset collection, documentation, content, design system, all pages, SEO/schema, and handoff docs.

**Architecture:** Astro static site with component-based layout system, JSON/TS content data files for reviews/services/FAQ, CSS custom properties for the design system, and JSON-LD schema injected per-page via BaseLayout. Forms use Formspree (no backend needed). Images live in `public/images/`.

**Tech Stack:** Astro 4.x, TypeScript, plain CSS (CSS custom properties), Formspree for contact form, JSON-LD for schema markup.
